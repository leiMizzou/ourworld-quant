# 参与贡献 · Contributing

感谢关注本项目!这是一个公开构建(build-in-public)的个人量化项目,欢迎以下形式参与:

## 可以做什么

- 🐛 **报 Bug / 提建议**:开 [Issue](https://github.com/leiMizzou/ourworld-quant/issues),说明问题或想法。
- 💡 **讨论方法论**:因子、回测陷阱、策略思路,欢迎在 Issue 里交流。
- 🔧 **提 PR**:修 bug、补文档、加因子/策略实现。请先开 Issue 对齐方向。

## 提 PR 的约定

1. Fork → 新建分支(`feat/xxx` 或 `fix/xxx`)→ 提交 → 发 PR。
2. 代码尽量带注释和最小可复现示例;涉及策略请附回测配置(扣费/滑点/区间)。
3. **不要提交任何行情数据文件、API token 或个人密钥**(见 `.gitignore`)。

## 重要原则

- 本项目所有内容**不构成投资建议**;PR/Issue 中请勿喊单或荐股。
- 回测结果请如实标注成本假设,拒绝"无成本暴利"式误导。

## 行为准则

友善、就事论事。我们讨论代码和方法,不做人身攻击。
