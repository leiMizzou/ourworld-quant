## 0. 先校准预期(别跳过这节)

个人量化和"在家点几下就躺赚"没有任何关系。把它当成一个**需要 6–18 个月才能跑通完整闭环**的工程 + 研究项目:

- **小资金的真相**:10万以内,即使年化做到 20%(已属优秀),一年也就 2 万。这个阶段**收益不是目标,跑通流程、积累可复盘的实盘记录(track record)才是目标**。职业化的敲门砖是"可验证的策略能力 + 完整的研究/实盘记录",不是这点本金的盈亏。
- **小资金的优势**:容量(capacity)几乎无限,可以做大资金做不了的低流动性、小市值、打新、可转债等策略;船小好掉头。**把这个优势当成你的护城河**。
- **真正的对手**:A股量化的对手盘是几千亿规模的量化私募(幻方、九坤、明汯等)+ 券商自营。你**不要去和他们拼速度(高频)**,要拼他们因为容量/合规而做不了的细分领域,以及中低频的逻辑型策略。
- **职业路径**:个人跑通 → 积累 1–2 年实盘曲线 → 投递量化私募/券商资管(研究员/策略岗),或自营/产品化。**面试看的是你的因子研究方法论和对回测陷阱的理解,不是你背了多少指标**。

## 1. A股市场结构认知(量化必须吃透的规则约束)

这些规则直接决定策略能不能做、回测要不要建模,务必在写第一行代码前理清:

| 规则 | 内容 | 对量化的影响 |
|---|---|---|
| **T+1** | 当日买入次日才能卖 | 无法日内回转(普通股票),日内策略受限 |
| **涨跌停** | 主板 ±10%,创业板/科创板 ±20%,ST ±5%,北交所 ±30% | 涨停买不进、跌停卖不出,**回测必须剔除一字板的不可成交** |
| **最小交易单位** | 买入 100 股一手;卖出可不足一手 | 小资金买高价股(如 茅台)一手就上万,组合分散度受限 |
| **集合竞价** | 9:15–9:25 开盘、14:57–15:00 收盘 | 很多策略在竞价撮合,需理解撮合规则与撤单时段 |
| **交易成本** | 佣金约万1.5–万3(最低5元)+ 印花税0.05%(2023年8月起,卖出单边)+ 过户费约0.001%(双边) | **回测必须扣费 + 滑点**,否则高换手策略全是假收益。具体费率以你券商为准 |
| **T+0 标的** | 可转债、部分跨境/债券/货币 ETF 支持 T+0 | 小资金做日内/网格的可选品种 |
| **杠杆/衍生品门槛** | 融资融券、股指期货、ETF期权均需 **近20交易日日均资产50万 + 6个月经验**等 | **10万阶段一律用不上,策略只能做"纯多头/做多"**,先不考虑对冲 |

**两个A股特有的坑要提前知道**:
- **小市值/微盘股因子**历史上在A股很强,但 2024 年初微盘股发生过流动性危机(量化集中赎回 + 雪球/DMA 连锁反应,微盘指数短期暴跌)。做小市值要把**流动性风险和极端回撤**建模进去,别只看历史年化。
- **退市/停复牌/除权除息**:回测数据必须做**前复权/后复权处理**,并包含已退市股票,否则严重**幸存者偏差**。

## 2. 程序化交易合规(2025 新规,做实盘前必看)

证监会《证券市场程序化交易管理规定(试行)》+ 沪深北交易所实施细则已于 **2025年7月7日正式实施**。要点:

- **报告义务**:程序化交易投资者需履行报告(报备)。个人通过券商 QMT/Ptrade 做程序化交易时,**报备一般由券商协助完成**,开通时按流程走即可。
- **高频交易认定标准**:单账户**每秒**申报+撤单合计最高达 **300 笔以上**,或单账户**全日**申报+撤单合计达 **20000 笔以上**。触发后有额外的经手费/流量费/撤单费等。
- **对个人的实际影响**:从试运行看,触发指标的主要是量化私募和券商自营,**中小个人投资者的正常交易基本不受影响**。你做中低频策略,离高频线差得很远,正常报备即可,**不要刻意去做高频**。
- 四类异常交易被明确:瞬时申报速率异常、频繁瞬时撤单、频繁拉抬打压、短时间大额成交——中低频策略自然规避。

> 结论:合规对你不是障碍,但**"做中低频"这个方向选择从一开始就被合规和资金双重锁定了**,这其实是好事,帮你避开和巨头的正面战场。

## 3. 技术栈搭建(你编程熟练,这部分最快)

### 数据源(免费起步,记住"不要单点依赖")
2025年8月 Tushare Pro 曾因机房纠纷**停运近一周**,引发量化圈震荡——教训是**任何单一数据源都会挂,要有备份**。

- **AkShare**:开源免费、品种最广,做日频研究够用;缺点是偶有缺失需自行清洗、实时接口约 500ms 延迟(不适合高频,你也不做)。
- **Tushare**:基础版积分制免费,数据结构化、稳定,财务/行情质量高;高级数据付费。
- **BaoStock**:免费,历史日线/复权数据干净,适合做基础数据校验和补漏。
- **建议组合**:`AkShare + Tushare 双源互备,BaoStock 校验`。后期上实盘再考虑付费高质量源(如 TickDB 等)。

### 回测框架(按用途选,别只用一个)
| 框架 | 定位 | 给你的建议 |
|---|---|---|
| **backtrader** | 事件驱动经典框架,直观 | **入门首选**,用来理解事件驱动、扣费滑点、订单撮合 |
| **vectorbt** | 向量化、极快 | 因子/参数批量快速验证,跑大规模回测 |
| **qlib (微软开源)** | AI/因子工程平台,内置 LightGBM/GRU/Transformer 等 | **职业方向核心**,中后期做多因子+机器学习用它 |
| **vnpy** | 全流程(研究→回测→实盘),模块化 | **国内实盘首选引擎**,接 CTP/券商接口 |

> 路线:`backtrader 打基础 → vectorbt 提速做因子 → qlib 上机器学习 → vnpy 接实盘`。

### 实盘接口(中后期才需要,先了解)
- **QMT(迅投)/ Ptrade(恒生)**:个人做程序化交易的两大主流终端。**国金证券等可 10万 门槛开通 QMT**(部分券商要 50万);Ptrade 多家券商门槛更低甚至无门槛,已有 50+ 券商接入。
- 注意:你资金 ≤10万,**正好卡在 QMT 10万门槛边缘**,开户前务必和客户经理确认当前门槛和佣金,优先选**低佣 + 支持低门槛 QMT/Ptrade** 的券商。
- **前期完全不用碰实盘接口**:先用聚宽/掘金的**在线模拟盘**或本地回测验证,跑稳了再开通。

### 在线平台(零搭建,适合最快上手验证想法)
- **聚宽 JoinQuant**:免费数据+研究+回测+模拟/实盘,因子库强,低频回测效率高,**适合做因子研究**。
- **掘金 MyQuant**:数据+回测+仿真+实盘一站式,自带实盘。
- **米筐 RiceQuant**:免费回测和模拟环境。
- 用法:**在线平台快速验证策略逻辑 → 本地框架做严肃回测和工程化**,两条腿走路。

### 工程化(你的编程优势所在)
- 环境:`conda`/`uv` 管理;`pandas / numpy / scipy / statsmodels / scikit-learn`。
- 数据存储:本地 `parquet + DuckDB`(轻量快),数据量大再上 `ClickHouse`。
- **从第一天就用 Git 管理代码和策略**,这是你职业化简历的一部分。

## 4. 知识体系(有交易经验,补"量化方法论"这块短板)

你懂市场,缺的是把直觉变成**可统计验证的方法论**。重点补:

- **因子投资**:单因子检验(IC/ICIR、分层回测)、因子正交化、多因子合成、组合优化、风险模型(Barra 思想)。
- **回测陷阱(最重要,面试必问)**:未来函数(look-ahead)、幸存者偏差、过拟合/参数挖掘、交易成本与滑点低估、样本内外划分、前视复权错误。
- **组合与风控**:仓位管理、最大回撤控制、止损逻辑、行业/风格中性化。

**书单(精读 2 本胜过泛读 10 本)**:
- 《因子投资:方法与实践》(石川等)——**A股本土因子实证,首推**,直接对标你要做的事。
- 《主动投资组合管理》(Grinold & Kahn)——组合管理理论经典(信息比率、基本法则)。
- 进阶:《Advances in Financial Machine Learning》(Marcos López de Prado)——机器学习做量化的避坑圣经,职业方向必读。

## 6. 第一个落地项目建议(本周就能开干)

**"小市值 + 基本面过滤 月度轮动"** —— 最适合小资金练手的入门项目:
1. 全A股按市值升序,排除 ST、停牌、上市不满1年、近期一字板;
2. 加基本面过滤(如剔除亏损、剔除高负债);
3. 选最小的 N 只(如20只)等权持有,每月底调仓;
4. 回测扣费扣滑点,**对比沪深300/中证2000基准**;
5. **务必把 2024 年初微盘股流动性危机那段单独拎出来看回撤** —— 这一步会让你真正理解"年化好看 ≠ 能拿得住"。

做完这个项目,你就完整经历了"取数→清洗→选股→回测→评估→复盘"的全流程。

## 7. 避坑清单(贴在显示器上)

- ❌ 上来就追高频/做T —— 拼不过私募,还可能触发合规线
- ❌ 回测不扣费不扣滑点 —— 高换手策略的"暴利"全是假的
- ❌ 参数调到回测最优 —— 过拟合,实盘必崩;坚持样本内外分离
- ❌ 单一数据源 —— Tushare 都会停运,要有备份
- ❌ 满仓单一策略上实盘 —— 先小资金、先模拟
- ❌ 只看年化不看回撤 —— 拿不住的曲线等于没有
- ✅ 每一个策略都写研究笔记 + Git 版本管理 —— 这是你的职业资产
