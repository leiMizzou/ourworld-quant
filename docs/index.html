// 这是什么

把量化这件事,完整地做给你看

不炫技、不喊单。目标是把个人量化从零到实盘的全过程,完整、透明、可复现地记录下来——代码开源、过程上站、心得做自媒体。

🧱

可复现

数据管道、回测框架、因子研究、策略,全部开源可跑,不藏关键步骤。

📖

透明

每个阶段的进展、踩的坑、改的错,都写进构建日志,真实呈现。

🎯

务实

适配小资金、中低频、纯多头的现实约束,不碰拼不过机构的高频。

// 为什么公开做

公开,本身就是策略的一部分

研究笔记加实盘曲线,是投递量化私募/资管最值钱的作品集。公开倒逼质量,顺手把个人品牌也做了。

📦 内容即资产

可验证的研究方法论 + 实盘记录,是职业化最硬的敲门砖,远比这点本金的盈亏重要。

🔁 公开倒逼质量

知道有人看,就不会糊弄回测、不会忽略交易成本、不会过拟合自欺。

📣 顺带做自媒体

站点 + 仓库 + 日志天然是自媒体素材,运营和个人 IP 一并完成。

🌍 回馈社区

国内个人量化的完整开源闭环不多,做一个真实、可参考的样本。

⚖️
现实约束反而帮我们聚焦。
10 万以内用不上融资融券/期权/股指期货(均需 50 万门槛),所以策略只能做纯多头、中低频;而高频拼不过几千亿规模的量化私募,还易触发程序化交易新规(每秒 300 笔 / 全日 2 万笔的高频线)。这把方向锁定在巨头因容量和合规做不了的细分战场——小资金的优势,不是劣势。
// 路线图

6 个月,跑通一个闭环

每个阶段都有明确产出。收益不是当前目标,跑通流程、积累可复盘的记录才是。

0

环境与数据管道 ✓ 完成

搭 Python 环境与 Git 仓库;AkShare + Tushare 双源互备、BaoStock 校验;建本地数据库(含已退市股票)与清洗模块。产出:可靠的本地数据管道。

1

回测引擎 + 复现经典策略 ✓ 完成

用 backtrader 搭含真实扣费/滑点/涨跌停不可成交的回测框架;复现一个经典策略,把回测陷阱踩一遍。产出:可信任的回测框架。

2

因子研究 进行中

单因子检验流水线(IC/ICIR/分层);测主流因子在 A 股的表现;多因子合成 + 组合构建 + 中性化。产出:多因子选股策略 + 研究笔记。

3

模拟盘验证 未开始

在聚宽/掘金模拟盘或 QMT 模拟环境实时跑 1–2 个月,定位滑点/冲击成本/延迟偏差。产出:回测与模拟一致的策略。

4

小资金实盘 未开始

选低佣券商开通 QMT/Ptrade;用最小可承受资金实盘,每日监控、记录、复盘。产出:真实的实盘曲线。
