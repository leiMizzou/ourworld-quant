02 — CONTEST

比赛不是装饰，是这个系统的主入口

01

统一起点

每个用户从同样的模拟本金开始，交易、持仓和收益都按同一套规则计算。讨论时少一点口水，多一点可比性。

02

真实行情

模拟盘价格来自本地 DuckDB 中同步的真实 A 股行情，成交价使用不复权价格，研究回测和模拟成交口径分开。

03

公开战绩

参赛者可以展示个人战绩页、战绩卡、持仓和近期成交。策略好不好，不只靠一句观点，而是看过程。

03 — DISCUSSION

把每一笔模拟交易变成可讨论的策略记录

POST
发起复盘

从比赛页一键生成带排名、收益、持仓和成交的复盘草稿。

COMMENT
围绕证据讨论

评论区可以讨论入场逻辑、仓位、风险和执行偏差。

PROFILE
沉淀个人策略轨迹

个人页会聚合战绩、交易、持仓和策略帖，方便持续观察。

这里鼓励什么样的互动

好的讨论不是“今天买什么”，而是把假设、条件、执行和结果拆开。你可以提出一个规则，也可以挑战别人的复盘，但需要围绕公开数据和模拟交易记录展开。

这不是荐股社区。所有内容用于技术研究、模拟训练和策略交流，不构成投资建议。
04 — RULES

赛制尽量贴近 A 股现实约束

交易规则

买入按 100 股整数倍，费用包含佣金、过户费和卖出印花税。当天买入不可当天卖出，账户页可以进入下一交易日释放可卖数量。

行情口径

模拟盘用不复权价格作为可成交口径，研究报告可以继续用后复权价格保持收益连续性。两套口径分开，避免把研究价格当成交价格。

比赛展示

公开榜单按模拟账户权益排序，个人页展示收益、持仓、成交和策略分享。排行榜是入口，复盘质量才是长期看点。

组合设计

登录后可以从真实行情生成反转、动量候选，也可以导入模型预测候选，先生成演练计划，再选择是否执行。
